# RC Arduino Boat

A simple Arduino Nano transmitter and reciever code for controlling the RC Boat for fish feeding.


## Parts

- 3x Servo motor (Rudder + Left and right feeder)
- 1x 20A ESC
- 2x NRF24L01 + PA
- 2x Rotary Encoder
- 2x Joystick button
- 2x 7.4V battery
- 1x Battery charger
- 1x 2S BMS
- 1x DC to DC converter
- 2x DC Motor 180
- 2x PVC Pipe
- XPS styrodur / styropor
- Acrylic glass shutter
- Glue
## Libraries / Includes

- #include <Rotary.h>
- #include <SPI.h>
- #include <RF24.h>
- #include <RF24_config.h>
- #include <nRF24L01.h>
- #include <printf.h>
